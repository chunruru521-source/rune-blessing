# rune-blessing

This repository is the starting point for the Rune Blessing project. As we add code, use this space to describe the goals of the project and how to get started.

## Local development

1. Clone the repository.
2. Create a feature branch from `main` before making changes.
3. Run your updates and commit frequently with clear messages.
4. Open a pull request when your work is ready for review.
